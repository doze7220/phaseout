# プリズムリンク専用カタルシス演出（UI・エフェクト・サウンド）の実装

## Goal
実装済みの「プリズムリンク」発生時において、プレイヤーの射幸心と快感を極限まで高める以下の3つの演出を実装します。
1. **7つのトライバルのカウントダウン（パチスロ風確定演出）**
2. **Pリンク通過時のレーザー膨張（太さの増加）**
3. **Pリンク専用の重低音SE（「どきゅぅん！！！」）とピッチ上昇**
※これらは最終的な「フェイズシフト（7色全接続）」に向けた儀式としての演出ですが、今回の実装スコープでは「7つ目の点灯」までとし、フェイズシフト自体のモード移行処理は後日実装とします。

## Proposed Changes

### 1. 演出定数とオーディオアセットの追加
*   **`config.js` (`EFFECT_MATH_CONFIG` 等)**:
    - Pリンク時のレーザーの太さの増加量（例: `PRISM_LASER_WIDTH_MULT`）などの演出定数を追加。
*   **`audioConfig.js`**:
    - `AUDIO_ASSETS.SE` に、Pリンク専用の重低音SE `PRISM_LINK_BURST` （「どきゅぅん！！！」というイメージ）を追加。現在はデータが用意されていないため、なにか適当なSEを割り当てること。

### 2. トライバル・カウントダウンUIの実装 (`ScreenEffects.js`)
*   **[NEW] 状態管理と描画ロジックの追加**:
    - 連鎖中に初めてPリンク（色の境界跨ぎ）が発生した瞬間に、画面上部（ヘッダー直下など）に7つの「トライバルアイコン（`symbol_1`〜`7`）」を横並びで半透明の黒塗りで表示します（出現時はグリッチノイズを描画）。
    - 跨いだ色の数（2色目、3色目…）に応じて、左から順にアイコンが灯る演出を行う。**「各トライバル上に透明＆大きくスケールアップしたトライバルアイコン → 加速度を付けて縮小しつつトライバルに重なる → 重なったタイミングでパーティクルを派手に撒き散らす＆一瞬白くなって元の陣営色になる」**というパチスロの確定ランプのような派手な点灯アニメーションで描画します。全フェーズを0.5秒以内に行ってください。
    - 連鎖終了時（`hideChainPopup` 等のタイミング）にグリッチノイズを描画し、フェードアウトさせます。

### 3. レーザー膨張とSEトリガーの実装 (`LaserEffect.js` & `effects.js`)
*   **[MODIFY] `animateLaserLevels` & `updateAndDraw`**:
    - レーザーの伝播処理において、`from` と `to` の宝石の色（`colorId`）が異なる場合を「Pリンク通過点」として検知します。
    - 通過点に到達した瞬間、描画中のレーザーの `lineWidth`（または加算合成の多重度）を段階的に太くし、視覚的なエネルギーの膨張を表現してください。
*   **[MODIFY] Pリンク通過時のイベント発火**:
    - 上記のPリンク通過を検知した瞬間に、`ScreenEffects` への「トライバル点灯の進行ステップ」の通知と、`SoundManager.playSE('PRISM_LINK_BURST', { playbackRate: ... })` の呼び出しを行います。
    - SEのピッチ（`playbackRate`）は、Pリンクを跨ぐごとに段階的に上昇させてください。
