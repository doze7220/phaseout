【出力フォーマット要件】

#### 凡例・項目定義
| 項目 | 説明 |
| ------ | ------ |
| 関数名 | クラスメソッドは クラス名#メソッド名 で表記 |
| 行番号 | 定義行の行番号 |
| 引数 | 関数の引数リスト |
| 戻り値 | 明示的な return がある場合のみ記述 |
| 呼び出し元 | 主な呼び出し元ファイル・関数 |
| **実行タイミング** | 【Phase Out特有】`beforeUpdate`, `collisionStart`, `afterUpdate` のいずれか、または `requestAnimationFrame` (描画ループ) のどこで発火するか。それ以外はイベント駆動など。 |
| **GameState** | 【Phase Out特有】`GameState` (例: `actualScore`, `life`, `currentExp` 等) を「参照(Read)」または「更新(Write)」しているか。していない場合は「なし」。 |
| 概要 | 処理の簡潔な説明。DOMアクセスを行っている場合はその旨も記載。 |
