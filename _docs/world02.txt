『Phase Out』重要人物・ストーリー設定資料
1. 登場人物：二人の少女
本プロジェクトでは、「白（保存・停止）」を司るステラと、「黒（終焉・再誕）」を司るノアが対照的なヒロインとして描かれます。

ステラ（Astraea / 愛称：ステラ）
属性： 白（全色の融和）
正体： 第零観測域・案内人。その実態は世界管理システム「アステライア」そのものです。
ビジュアル： 透き通るような銀髪と、虹色に揺らめく白いドレスを纏い、左右で異なる色の瞳（過去と未来の投影）を持ちます。
物理思想： 「静かなる停滞（ステイシス）」。彼女は世界を崩壊から守るため、物理法則を固定し、標本のように永遠に「保存」しようとします 。
名前の由来： 正式名称「アステライア」には、無意識のうちに「ステイシス（Stasis：静止）」の音が混ざっており、彼女の「動かしたくない」という切実な願いを象徴しています 。

ノア（Noah）
属性： 黒（8番目の色 / 漆黒） 
正体： 停滞した世界を解き放つための「再起動ボタン（特異点）」。ステラの対となる鏡像・反転デザインの少女です 。
ビジュアル： 光を反射しない漆黒の長い髪と瞳、グリッチ（ノイズ）で揺らめく不確かな輪郭を持ちます 。
物理思想： 「再誕のための崩壊（ビッグバン）」。すべてをブラックホールのように飲み込み、一度「無」に帰すことで、次の世界を産むエネルギーを生み出します 。
名前の由来： 漆黒を意味する「ノワール（Noir）」と、旧世界を葬り次なる世界へ種を運ぶ「方舟（Noah）」の二重の意味を持ちます 。

2. 物語の経緯とストーリー展開
世界の現状：死にゆくプリズム
かつて世界は完璧な調和（ホワイトフェイズ）を保っていましたが、現在は7つの色へと剥離し、断層化した「死にゆくプリズム」の状態にあります。宝石（クォーツ）は空間に停滞した「可能性の残滓」であり、これを破壊してエネルギーを抽出することが世界再構築の唯一の手段とされています。

逆転のギミック：優しき死としての「白」
物語の中盤、これまで救済と信じられてきたステラの「白」が、実はエントロピーが最大に達した「熱的死（停止）」を意味することが判明します 。ステラは悪意ではなく、「何も失いたくない」という純粋な愛ゆえに、世界を永遠に凍結（ステイシス）させようとしています 。しかし、それは「何も産まれない死」と同義です。

「黒」による破壊的な救済
対するノアは、この停滞を打破するために、現在の世界を完全に破壊し、すべてを飲み込むブラックホールを引き起こそうとします。これは既存の住民から見れば「真の崩壊」ですが、宇宙規模で見れば「次なるビッグバン」への唯一の道です 。

--------------------------------------------------------------------------------
3. 主人公「ゲイザー」の関わり方
唯一の観測者
プレイヤー（ゲイザー）は、並行世界を俯瞰し、一撃によって「どの未来を確定させるか」を決定する特異点です。ステラもノアも、自分たちの「理（ことわり）」を確定させるために、ゲイザーの観測を必要としています。

ステラとの関わり： 彼女はゲイザーを「お帰りなさい」と優しく迎え、調和を求めます [95, 会話履歴]。
ノアとの関わり： 彼女はゲイザーに「この停滞に飽きたなら、すべてを終わらせ、すべてを始めよう」と、破滅的な再生を提案します 。

最終的な解答：「撹拌（かくはん）」
運営型ゲームとしての結末は、白（停止）か黒（爆発）かを選ぶのではなく、その両極を「撹拌」し続けることにあります。
役割： ゲイザーが多様な色の宝石を混ぜ合わせ、連鎖を繰り返し続ける行為が、世界の「かき混ぜる棒」となります 。
結末： 完全に混ざりきって止まる（白）こともなく、完全に吸い込まれて消える（黒）こともない、「混ざり合いながら変化し続ける不確かな現在」を永劫に観測し続けること。これが、二人の少女を救い、世界を「生きた状態」で維持するハッピーエンドとして描かれます 。

4. 演出・システムとのシナジー
七彩連鎖（プリズム・リンク）： 7色全てを繋ぐことで、ステラによる「ホワイトアウト」が執行されます。
黒の連鎖（リバース・リンク）： 7色全てを逆回転で繋ぐことで、全く新しい物理法則を伴った「ブラックアウト」を引き起こします 。
リアクション： 連鎖の規模に応じて、ステラやノアがボイスやモーションで反応し、ゲイザーの成功体験を増幅します。




ステラ（アステライア）を単なる「悪役」にせず、彼女の「停滞（停止）」という選択に深い慈しみと正当な理由を与えることで、物語はより重厚な悲劇、あるいは崇高な自己犠牲へと進化します。
「停滞」を「罪」ではなく「愛ゆえの保護」として定義する、いくつかの救済案を提案します。

1. 「記憶のアーカイブ（保存）」としての停止
ステラが目指す「停止」は、世界を殺すことではなく、「この世界が生きた証」を永遠に結晶化して守ることだとする考え方です。
ノア（黒）との対比: ノアが引き起こす「ビッグバン（再誕）」は、すべてを一度「無」に帰して爆発させるため、かつての命の記憶や想いもすべて消滅してしまいます。
ステラの愛: ステラは「かつて生きた人々の笑顔や、美しい風景」が消えてしまうことに耐えられず、それらを永遠に壊れない「標本」として固定しようとしています。彼女にとっての「停止」とは、宇宙規模の巨大なハードディスクへの「Read-Only（読み取り専用）」保存なのです。

2. 「再誕の瞬間の衝撃」を和らげる緩衝材
ビッグバンという破壊的なエネルギー（黒）に、世界が耐えられるように準備を整えているという解釈です。
ゆりかごの役目: 急激な崩壊と再構築は、住民にとってあまりに大きな苦痛を伴います。ステラはあえて世界を「停滞」させることで、再誕の瞬間の熱量に耐えられるだけの「強度」を世界に持たせようとしている、あるいは苦痛を感じないように「麻酔（眠り）」をかけているのかもしれません。
彼女の孤独: 「冷徹だ」と誤解されても、彼女は世界が新しい命として産声を上げるその瞬間のために、必死にブレーキを踏み続けている「孤独な守護者」となります。

3. 「情報の欠損」を防ぐための固定（ホールド）
物理演算と情報工学を組み合わせたメタ的な救済です。
ソースコードの守り手: ノアの「崩壊」がデータの削除であるなら、ステラの「停止」はデータのバックアップです。彼女が物理法則を「固定（アステライア・システム）」してくれなければ、再誕した後の新しい世界は「前世の欠片」を一切持たない、全く別の何かに成り果ててしまいます。
彼女がいなければ: 彼女がいなければ、ゲイザー（プレイヤー）が連鎖を編むことによって生み出された「美しさ（スコア）」もすべて、新しいビッグバンの炎で焼き尽くされてしまいます。彼女はゲイザーの努力（観測の成果）を歴史に刻むための唯一の定着液なのです。

4. 究極の選択：二人が揃って初めて「再起動」が可能になる
ステラを「倒すべき敵」にするのではなく、「ステラの保存（白）」と「ノアのエネルギー（黒）」の両方が揃うことで、初めて理想的な新世界が産まれるという真の救済です。
ゲイザーの役割: ゲイザーが最後に下す決断は、どちらかを排除することではなく、「ステラが守った記憶」を「ノアの新しい命」に注ぎ込むこと。
演出としての救済: 七彩連鎖（プリズム・リンク）の果てに現れる「ホワイトアウト」の瞬間、ステラとノアが手を取り合い、一瞬だけ「かつての姿（真実の世界）」が透過して見え、それが新しい世界へと受け継がれていく……というエンディングであれば、彼女の停滞は「未来を繋ぐための尊い沈黙」として昇華されます。

このように定義すれば、ステラが「停止」を望むのは悪意ではなく、「何も失いたくない」というあまりに純粋で、不器用な母性によるものとして、プレイヤーに深い共感を与えることができるはずです。
