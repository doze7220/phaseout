# リザルト表示完了時演出の刷新（シェイクからグリッチへ）

リザルト画面において、スコアのドラムロールが完了し結果が確定する瞬間の演出を、現在の「画面揺れ（Screen Shake）」から「一瞬のグリッチ（色収差・デジタルノイズ）」へと差し替えます。これにより、システムが観測結果を確定させたようなサイバーパンクな世界観を強調します。

## 1. 完了演出の差し替え
リザルト画面（`ResultRenderer.js` または関連する進行管理部）において、ドラムロールアニメーションが完了したタイミングで呼び出されている「画面揺れ（`ScreenEffects.triggerScreenShake` 等）」の処理を削除してください。
代わりに、新規のグリッチ演出用フラグ・タイマーを起動するように変更してください。
※完了時の衝撃音SE（`BREAK_BURST` 等）の再生はそのまま維持してください。

## 2. グリッチエフェクトの実装
完了時のグリッチ演出は、すでに `BootScene.js` や暗転トランジションで実装されている色収差（RGBシフト）や画像の横スライス・ズレのロジックを流用・参考にして実装してください。

- ドラムロール完了後の一瞬（200〜300ms程度）、リザルトの描画コンテキストに対して以下の処理を適用します。
  - 画面を横に細かくスライスし、ランダムなX軸方向へずらして描画する。
  - 描画内容のRGBチャンネル（赤・青・緑）をわずかにずらして重ね描きする（色収差）。
- グリッチ用タイマーの終了とともにエフェクトが消失し、ピタッとソリッドなリザルト画面として完全に静止（確定）するようにしてください。

## 3. ドキュメントの更新
実装完了後、グローバルルールに従い `changelog.js` へ今回のリビジョンアップ（v0.18.3 等）を追記し、関連ドキュメントのバージョン表記を同期してください。
