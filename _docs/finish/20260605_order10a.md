あなたは優秀なゲームエンジニアです。
パズルゲーム『Phase Out: Cluster Stirring』において、UI改修のStep1およびStep1aとして、「LIFE/EXPゲージの同心矩形化」「連続レベルアップ演出」「Lv表記のスリム化」、そして「各獲得数値のフローティング表記」を実装します。
`PROJECT_ARCHITECTURE.md`と`PROJECT_FUNCTION_INDEX.md`のルールに従い、DOM操作とロジックの責務を分離して実装してください。

【対象ファイル】
・config.js (表示用GameState変数の追加)
・logic.js (内部経験値の即時計算、ポップアップ呼び出し、旧オーラ演出削除)
・ScreenEffects.js (ゲージ改修、連続レベルアップ演出、フローティングUI生成)
・effects.js (フローティングUI用API追加、旧オーラ演出削除)
・style.css (フローティングテキスト用アニメーション追加、Lv表記刷新、旧オーラ削除)
・index.html (不要なプログレスバーDOMの削除)

【実装要件】
1. 既存のパネル（オーラ）エフェクトの完全削除
・「一番多く破壊している宝石の色でヘッダーがフラッシュする」演出、および `--aura-color` に関連するCSSやJSロジックを完全に削除してください。

2. Lv表記デザインの刷新とスリム化 (`index.html`, `style.css`)
・現在配置されている「撹拌Lv」表記内のプログレスバー要素を完全に削除してください。
・コンテナの横幅をスリム化し、角丸の四角（border-radius: 6px等）に変更してください。

3. 各獲得数値のフローティング表記（Step1a）
・`style.css` に、文字が上にフワッと浮かび上がって消えるポップアップアニメーション（例: `.floating-text`）を定義してください。色は以下の3種類です。
　- 減少LIFE（ダメージ）: オレンジ（#FF9500等、ゲージ色と連動）
　- 回復LIFE（ヒール）: 緑（#00FF00等）
　- 獲得経験値（EXP）: 青（#00BFFF等）
・`effects.js` および `ScreenEffects.js` に `showFloatingNumber(text, type, delay)` を追加してください。要素は「Lv表記ボックスのすぐ下あたり」に絶対配置で生成し、アニメーション終了後にDOMから削除するようにします。
・`logic.js` に組み込みます。
　- タップ時のLIFE減少時: `showFloatingNumber('-' + damage, 'damage', 0)` を呼び出す。
　- 破壊によるLIFE回復時: `showFloatingNumber('+' + heal, 'heal', 0)` を呼び出す。
　- 破壊によるEXP獲得時: `showFloatingNumber('+' + exp + ' EXP', 'exp', 500)` を呼び出し、**0.5秒（500ms）遅延させてから表示**してください（回復表示との被りを防ぐため）。

4. 状態定義と内部計算 (`config.js`, `logic.js`)
・`GameState` にアニメーション追従用の `displayExp: 0` と `displayLevel: 1` を追加してください。
・`logic.js` で経験値を獲得した際、アニメーションを待たずに内部の `exp` と `level` を即座に計算・確定させてください（whileループで複数レベルアップにも対応すること。計算式は今回は変更しません）。

5. LIFE・EXPゲージの同心矩形化と連続レベルアップ (`ScreenEffects.js` - `GaugeManager`)
・SVGのLIFEゲージの「内側」に、同じ形状の「EXPゲージ」パスを追加し、`(displayExp / nextLevelExp)` の割合で下から上へ満ちるようにしてください。
・描画ループ内で `displayExp` を実際の `exp` に向けて猛スピード（約0.5秒でMAX）で増加させます。
・`displayExp >= nextLevelExp` になった瞬間、レベルアップ演出（ポップアップとゲージフラッシュ）を発火させ、`displayExp` から `nextLevelExp` を引き、`displayLevel` を+1する処理を、実際の `exp` に追いつくまで連続で繰り返すようにしてください。

【遵守事項】
・UIアニメーションと内部ロジック（GameState確定）の分離を意識してください。
・フローティングテキストは表示が重なってもレイアウトが崩れないよう、position: absolute を適切に使用してください。
