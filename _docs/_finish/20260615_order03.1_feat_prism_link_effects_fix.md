# プリズムリンク専用カタルシス演出（完全版）の実装

## Goal
「プリズムリンク」発生時において、プレイヤーの射幸心と快感を極限まで高める以下の演出を実装します。
1. **プリズムゲージのパチスロ風スタンプ点灯（黒い板を廃止したソリッドなUI）**
2. **「プリズム深度（Prism Depth）」による重複排除と、極太レーザーへの膨張**
3. **Pリンク専用の重低音SE（「どきゅぅん！！！」）とピッチ上昇**
4. **スコア確定と同時（`finalizeDestruction` 連携）のグリッチ消去**

## Proposed Changes

### 1. 演出定数とレイアウトのマジックナンバー分離
*   **`LayoutConfig.js`**: `LAYOUT_CONFIG` 内に `PRISM_GAUGE` を新設し、ゲージ全体のYオフセット、アイコンサイズ、マージン等を定義。
*   **`config.js` (`EFFECT_MATH_CONFIG`)**: `PRISM_LINK` を新設し、スタンプ落下時間、最大スケール、白フラッシュ時間、レーザー膨張倍率等を定義。
*   **`audioConfig.js`**: `AUDIO_ASSETS.SE` に重低音SE `PRISM_LINK_BURST`（「どきゅぅん！！！」）を追加。

### 2. 「プリズム深度」による状態管理と発火制御 (`LaserEffect.js` & `ScreenEffects.js`)
*   **[NEW] プリズム深度の導入**:
    - `LaserEffect` 側で、伝播中の各レーザー線（`lightLines`）が「起点から何色跨いだか」を示す `prismDepth` を保持。
    - `ScreenEffects` 側（または `GameState` 等適切な場所）に、今回の連鎖で到達した「全体の最大プリズム深度 (`maxPrismDepth`)」を管理するステートを設ける。
*   **[MODIFY] 発火判定**:
    - 各レーザーがPリンクを跨ぎ `prismDepth` が加算された際、それが全体の `maxPrismDepth` を上回った（新深度へ到達した）**最初の1回目のみ**、`ScreenEffects` へゲージ点灯ステップを進める通知を送り、SE `PRISM_LINK_BURST`（深度に応じたピッチ上昇付き）を鳴らす。
    - レーザーの太さ（`lineWidth` 等）は、全体の最大深度ではなく、個々の線が持つ `prismDepth` に応じて個別に太く（膨張）描画する。

### 3. プリズムゲージ（UI）の描画とスタンプ演出 (`ScreenEffects.js`)
*   **[NEW] ソリッドなデザインとパチスロ風スタンプ演出**:
    - 従来の「黒い板の背景」を廃止し、ワイヤーフレームまたは漆黒のシルエットのアイコンをパズル領域/ヘッダー空間に直接浮かべる。
    - ステップ進行通知を受け取ると、対象のトライバル（アイコン）に対して以下の「スタンプ確定演出」を実行。
        1. 透明＆巨大化からの加速度付き落下（縮小）。
        2. 重なった瞬間に `effects.spawnBurstSparks()` を呼び出しパーティクルを散らす ＆ `effects.triggerScreenShake()` で盤面全体を揺らす。
        3. 一瞬の白飛び（フラッシュ）後、内部からグロウ（発光）を伴う金属的でリッチな陣営色として実体化させる。
*   **[MODIFY] スコア確定と同時の消去**:
    - 連鎖が終了しスコアが確定するタイミング（`hideChainPopup` 呼び出し時、または `finalizeDestruction` 内）に連動して、プリズムゲージ全体をフェードアウトではなく、**「グリッチノイズと共に瞬時にデータへと変換され消滅する（ショートする）」**ようなキレのある演出で破棄・非表示化する。
