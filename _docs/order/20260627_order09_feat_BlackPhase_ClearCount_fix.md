１．ブラックフェイズクリア時処理
ブラックフェイズクリア時にも、ホワイトフェイズクリアカウントを＋１すること。
ブラックフェイズはホワイトフェイズを経由しないと突入できない。
だが今のままでは、ホワイトフェイズをプレイしたことにならない。
これでは問題がある。

２．ブレイクゲージ獲得ポイントの補正
ブラックフェイズクリア回数を元に、ブレイクゲージ増加ポイントの補正を行う。これはホワイトフェイズと同等の処理を行う。計算式は以下の通り。

最終獲得ポイント ＝ 獲得ポイント * 0.8^クリア回数

この`0.8`はマジックナンバーにせず、ホワイトフェイズと同様にconfig.jsにて定義すること。ホワイトフェイズの計算式を確認し、問題がないように実装する。

３．デバッグウィンドウの表示
これらがデバッグウィンドウで確認できるようにすること。
現在「Ｒゲージ」と表示されている箇所である。
これを「Ｂゲージ」に修正。
そして表示内容を「Ｓゲージ」と同等にすること。
これらの値はＳゲージと同様に、リアルタイムで反映するようにすること。

以上の追加指示で、不明点・問題点があれば実装作業をやめ、確認を取ること。
問題がなければ実装を頼む。
