# 実装指示書：ブラックホール本体（引力・吸い込み・無限チェイン）の実装

## 目的
ブラックフェイズ中、画面中央の特異点（ブラックホール）が巨大な引力を発生させ、盤面の全宝石を吸い込んで破壊（無限チェイン）し続ける、物理エンジンへのアトラクター（引力）ロジックを実装する。

## 変更対象ファイル
*   `js/core/logic.js` (引力の適用、吸い込み判定、無限チェインのフック)
*   `js/core/effectConfig.js` (引力・吸い込み判定用のコンフィグ追加)

## 実装手順

### 1. 引力および吸い込み用コンフィグの追加 (`effectConfig.js`)
*   `BLACK_PHASE_EFFECT_CONFIG.BLACK_HOLE` 内に以下のパラメータを追加する。
    `ATTRACTOR_FORCE: 0.005` // 毎フレーム宝石に加わる引力の強さ（質量に応じたスケーリングが必要）
    `EVENT_HORIZON_RADIUS: 40` // 吸い込み判定となる事象の地平線（中心からの距離）

### 2. 引力（アトラクター）の適用 (`logic.js`)
*   アーキテクチャルールに従い、`beforeUpdateHandler` の内部にて以下の処理を追加する。
*   現在のフェイズが `PHASE_BLACK` の場合のみ実行する。
*   画面中央座標 `(APP_WIDTH / 2, APP_HEIGHT / 2)` を特異点の中心点とする。
*   `GameState.GEMS` のすべての宝石（Body）に対し、中心点へ向かうベクトルを計算し正規化する。
*   `Matter.Body.applyForce(gem, gem.position, force)` を使用して、中心へ向かう引力を毎フレーム加算する。（`force` は `ATTRACTOR_FORCE` に `gem.mass` を乗算するなどして調整すること）

### 3. 吸い込み（破壊）判定と無限チェインの接続 (`logic.js`)
*   引力を適用するループと同じタイミングで、宝石が特異点の中心から `EVENT_HORIZON_RADIUS` 以内に入ったかを判定する。
*   判定を満たした宝石は「ブラックホールに吸い込まれた」とみなし、吸い込まれた宝石の配列 (`swallowedGems`) に一時格納する。
*   フレームの最後に `swallowedGems` が1つ以上存在する場合、通常のタップ連鎖と同様に `finalizeDestruction` を呼び出し、スコア加算と破壊処理を行う。
*   ※ブラックフェイズ中は途切れることのない「無限チェイン」として扱うため、吸い込み発生時は内部のチェインカウントを加算し続ける処理を講じること。
