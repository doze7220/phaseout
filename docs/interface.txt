ソース資料に基づき、連鎖物理パズル『Septet Hexa（セプテット・ヘキサ）』およびその世界観『Phase Out（フェイズ・アウト）』のゲーム画面UIレイアウト案を具体化します。
このUIは、プレイヤー（ゲイザー）が世界を観測するためのデバイス「プリズム・レンズ」としてのメタ設定を反映しています
。
１．メイン画面のレイアウト構成（縦画面想定）
スマートフォンでの操作性を考慮し、画面を大きく3つのエリアに分割します。
① 上部：ステータス・情報エリア
インフレ・スコア表示（中央上部）: 指数関数的に跳ね上がる数値をダイナミックに表示します。加算時はドラムロール演出や急激なカウントアップを行い、数値の「重み」を視覚化します
。
陣営勢力ゲージ（最上部細枠）: 7陣営の勢力図をリアルタイムでバー表示。現在どの色が優勢か、翌週の物理法則がどうなりそうかを常に示唆します
。
ステージ情報 / 残り時間（右上）: 7つの階層（レイヤー）のどこを観測しているかを表示します
。
② 中央：メイン物理盤面「容器（Vessel）」
物理シミュレーション領域: Matter.js等の物理エンジンで制御された400x600程度の領域です
。7色の宝石（クォーツ）が物理演算でギチギチに積み上がります
。
背景「未確定のノイズ」: 宝石がない部分は砂嵐のようなノイズですが、宝石を破壊した一瞬だけ背後の「真実の世界」がフラッシュバックして見えます
。
連鎖エフェクト: タップ地点から同色へ光が伝播する際、宝石同士を「光の線」で結び、ネットワークを可視化します
。
③ 下部：アクション・キャラクターエリア
ナビゲーター「ステラ（アステライア）」: 画面の左下または右下に配置。連鎖の規模に応じて、ボイスや表情のカットイン（リアクション）でプレイヤーの成功体験を増幅します
。
物理属性インジケーター: 現在の世界状態（拡散・圧縮・崩壊）をアイコンと色で表示。どの戦略が有効かを一目で判別させます
。
広告視聴済みチケット・スロット: 最大5枚のストック状況を可視化。アツい瞬間に即座にタップして消費できるよう、押しやすい位置に配置します
。
--------------------------------------------------------------------------------
２．主要UIコンポーネントの詳細設計
コンポーネント
具体的な表示内容・演出
プリズム・リンク・トリガー
7色同時連鎖（プリズム・リンク）が成立しそうになると、画面外縁が虹色に発光。成立の瞬間に画面中央にステラのカットインと共に「WHITE OUT」の文字が爆発します
。
コンボ・カウンター
画面中央付近。破壊数 n に応じて指数関数的に巨大化し、スコアの「桁」が跳ね上がる瞬間に閃光を放ちます
。
属性オーバーライド・ボタン
チケットを消費して盤面の属性（三すくみ）を強制的に書き換えるための特殊ボタンです
。
キャラ・リアクション吹き出し
大連鎖時に「止まらない！」「やばい！」などのセリフが、スコアと同期して画面を覆うようにポップアップします
。
--------------------------------------------------------------------------------
３．視覚的フィードバックと「現象」の演出
光の減衰と強調: 通常時は静謐な暗闇ですが、連鎖が起きた場所から波紋のように光が広がり、盤面を「観測」している感覚を強調します
。
物理的衝撃の可視化: 宝石が砕ける際、画面全体が揺れる「スクリーンシェイク」を発生させ、特に大連鎖時にはUI自体が数値の重みに耐えきれず「ひび割れる」ようなエフェクトを検討します
。
情報の階層化: 「色（論理層）」の繋がりはクリアな光の線で、「形（物理層）」による崩壊は重厚な破片エフェクトで描き分け、プレイヤーの認知をハックします
。
このレイアウト案は、**「わかりやすさ（色での連鎖）」と「予測不能な興奮（物理崩壊）」**を両立させ、ゲイザーを世界観に没入させるためのインターフェースとして機能します
。
